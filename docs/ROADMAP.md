# Roadmap — Oficio

## Fase 1 — MVP (4-6 semanas)

- [ ] Landing page con propuesta de valor
- [ ] Listado de prestadores por ciudad y rubro
- [ ] Perfil de prestador con reseñas
- [ ] CTA a WhatsApp con mensaje pre-cargado
- [ ] Página de planes (sin pago real aún)

## Fase 2 — Beta (2-3 meses)

- [ ] Registro de prestadores
- [ ] Pago de suscripción con MercadoPago
- [ ] Panel del prestador (ver mis contactos)
- [ ] Sistema de reseñas verificadas (post-trabajo)
- [ ] Mapa interactivo con Mapbox
- [ ] Login con teléfono (Supabase Auth)

## Fase 3 — V1.0 (6 meses)

- [ ] PWA / App móvil
- [ ] Notificaciones push (OneSignal)
- [ ] Chat interno o redirect inteligente a WhatsApp
- [ ] Panel de admin (gestión de prestadores y suscripciones)
- [ ] Expansión: Neuquén, Cipolletti, Centenario
